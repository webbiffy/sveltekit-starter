# SvelteShop

SvelteShop is a personal project I'm using to test and practice Svelte. It's a casual playground where I can try out features, experiment with components, and gradually build a small shop-style app as I learn. It's not meant for production use, just for my own learning.

To get started, install the dependencies with npm install, and then start the dev server with npm run dev.

My main goals with this project are to get comfortable with Svelte components, stores, and routing, to practice building a small e-commerce style UI, and to experiment with what SvelteKit has to offer.
